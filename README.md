# 🗳️ ElectWise

ElectWise is an AI-powered web application that helps users understand the election process in a simple, interactive, and engaging way.

## 🚀 Features

* 🤖 AI Chatbot (Gemini API) for election queries
* 📍 Step-by-step election process guide
* 📅 Interactive election timeline
* ❓ FAQ section for common questions
* 🎯 Beginner-friendly UI

---

## 🛠️ Tech Stack

* React + Vite
* JavaScript
* Gemini API (Google AI)
* CSS

---
